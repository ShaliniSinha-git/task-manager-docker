# Task Manager API (Spring Boot + Docker)

A RESTful Task Manager application built using Spring Boot, PostgreSQL, and Docker.

## 🚀 Tech Stack
- Java 21
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Docker
- Docker Compose

## 📦 Features
- Create Task
- Get All Tasks
- Get Task by ID
- Update Task
- Delete Task
- Global Exception Handling

## 🐳 Run With Docker

```bash
docker-compose up --build

## 📡 API Endpoints
GET /tasks
POST /tasks
PUT /tasks/{id}
DELETE /tasks/{id}
